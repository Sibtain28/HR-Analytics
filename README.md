## Business Problem

Organizations struggle to retain top talent and optimize hiring decisions without clear visibility into workforce trends. This project analyzes HR data to uncover patterns in employee attrition, recruitment efficiency, and performance metrics — enabling HR teams to take proactive, data-backed decisions.

**Core Business Question**

> What factors drive employee attrition, and how can recruitment and performance data be used to improve workforce retention?

**Decision Supported**

> This dashboard enables HR managers to identify at-risk employees, optimize hiring pipelines, and allocate resources toward high-performance teams.

## Dashboard Views

| View | Description |
|---|---|
| **Attrition & Turnover** | Tracks employee exit trends, attrition rate by department, tenure, and demographics |
| **Recruitment & Hiring Pipeline** | Visualizes hiring funnel, time-to-hire, and source effectiveness |
| **Performance & Productivity** | Analyzes performance ratings, promotion trends, and team productivity metrics |
